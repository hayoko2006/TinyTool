功能模块

🔎

系统自检

8 项健康度检测：CPU、内存、磁盘、网络、安全、服务、更新，生成评分报告。

🔄

软件源管理

阿里云/清华/中科大/腾讯/华为 5 大镜像源一键切换，自动备份与恢复。

💾

存储管理

数据盘自动挂载(fstab 持久化)、LVM 在线扩容、磁盘占用分析、大文件排查。

⚙️

系统管理

一键系统更新、时区设置+时间同步、Swap 管理、缓存清理、服务管理。

🌐

DNS 管理

5 种 DNS 切换、解析测试、延迟测速、一键恢复默认配置。

🛡️

网络管理

SSH 端口修改、防火墙管理、网络测速(带宽/热门网站/节点延迟)、邮件服务检测、端口扫描。

🔑

SSH 管理

配置查看、一键安全加固、禁止密码登录、密钥管理、fail2ban 防爆破。

🏠

宝塔管理

一键安装宝塔面板、信息查看、密码重置、数据盘挂载到 /www、卸载。

🔌

Caddy 管理

官方源安装、状态查看、启停控制、Caddyfile 编辑、一键卸载。

🐳

Docker 管理

官方脚本安装+镜像加速、数据目录迁移、分层清理、Compose 管理。

📈

进程监控

CPU/内存 TOP 20、按名称/端口查找进程、kill 结束进程、进程树查看。

📋

日志工具

系统日志、认证日志、Web 日志、dmesg、启动日志、关键词搜索、清空。

⏰

定时任务

查看/新增/删除 crontab 任务、编辑 crontab、服务状态、常用模板。

📤

配置导出

系统配置一键导出（网络/SSH/防火墙/定时任务/磁盘/软件包），打包 tar.gz。

🏠

1Panel 管理

一键安装、面板信息查看、密码重置、端口修改、状态/重启、数据盘迁移、卸载。
